🎨 Emotion Theme
Custom WordPress theme built with Advanced Custom Fields (ACF Pro), modular template architecture, AJAX functionality, and responsive UI components.

⚡ Features
Custom WordPress theme development
Modular architecture with template-parts
Advanced Custom Fields (ACF Pro)
Dynamic theme switcher using cookies
Responsive burger navigation menu
AJAX modal windows
Swiper.js gallery slider
FAQ accordion component
Contact Form 7 integration
Theme Options Page via ACF
Optimized asset loading
Lazy-loaded images
Custom image sizes
Responsive layout
Performance optimizations

⚙️ Installation
1. Clone repository
git clone https://github.com/YanaShumylo/emotions
2. Move theme to WordPress themes folder
wp-content/themes/
3. Install dependencies
npm install
4. Activate theme in WordPress admin panel
Appearance → Themes
5. Install required plugins
Advanced Custom Fields Pro
Contact Form 7
